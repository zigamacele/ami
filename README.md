## Features

- Authentication with Anilist.
- Users can add, remove and edit their media lists.
- Users can add media to their favorites.
- Users can check their notifications.
- Users can search Anilist database.
- In depth Sesonal Anime Section.
- Account & List Settings.

## Technologies used

- Tauri
- NextJS
- OAuth2
- GraphQL
- TypeScript
- TailwindCSS
